MOST USED

  • Codex — AI coding agent that can read, edit, and work with codebases autonomously. It helps product designers turn Figma designs into working UI, build prototypes, and implement small design changes without relying on developers

    AI Agent / Build

  • Xcode — a development environment for building iOS apps. It is useful for testing real app behavior and turning prototypes into products.

    Build / Prototype

  • GigaType — the fastest voice keyboard for the Russian language. It works locally, without internet, and stays fully private

    Voice Input

  • Are:na — the best reference resources for designers, with a much lower amount of bad design noise than most platforms. The interface is not always the most convenient, but the quality of inspiration makes it worth using

    Inspiration

  • After Effects — useful for designers who need to make an idea feel vivid, persuasive, and easy to sell through animated interfaces, prototypes

    Prototype / Motion

  • X — it can be a constant source of ideas for projects, while also helping them stay aware of what is happening in the fast-changing world of design

    Trends / Ideas

  • Hermes Agent — AI agent for automating tasks, researching information, and supporting workflows. It is useful because it works as a Telegram bot, making it easy to access help, delegate repetitive work

    AI Agent

  • SF Symbols App — Apple’s official library of system icons
